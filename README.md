
# Marvel Characters

## Description
 Sample app based on Marvel's API, using MVVM design pattern
taking advantage of the jetpack tools:

- Navigation
- ViewModel
- Coroutines
- ViewPager2
- Android X
- Room
- Lifecycle

#### Third party tools:

- Glide
- Connector
